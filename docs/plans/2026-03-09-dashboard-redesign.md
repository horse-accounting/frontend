# Dashboard Redesign

**Огноо:** 2026-03-09

## Зорилго

Дашбоардыг визуал сайжруулж, `nasaarAngilal` өгөгдлийг нэмж, бүлэг/үүлдэр/хүйсийн мэдээллийг илүү эвтэйхэн харуулах.

## API

Одоогийн `GET /api/stats` endpoint бүх шаардлагатай өгөгдлийг буцаадаг. Шинэ endpoint шаардлагагүй.

## Шинэ library

- `@ant-design/charts` — зөвхөн Pie (donut) chart-д ашиглана

## Layout

### 1-р мөр: Stat card-ууд (өөрчлөлтгүй)

4 card: Нийт адуу, Нийт үүлдэр, Нийт бүлэг, Уралдсан

### 2-р мөр (3 баганатай)

| Багана | Агуулга | Хэрэгжүүлэлт |
|--------|---------|---------------|
| Зүүн (xs:24 lg:8) | Хүйсээр | Donut chart (Pie innerRadius). Эр: #1890ff, Эм: #eb2f96. Төвд нийт тоо |
| Дунд (xs:24 lg:8) | Сүүлд нэмэгдсэн | Одоогийн List хэвээр |
| Баруун (xs:24 lg:8) | Шилдэг адуунууд | Одоогийн List хэвээр |

### 3-р мөр (бүтэн өргөн)

**Насаар ангилал** — ШИНЭ

Ant Design Table + Progress bar:
- Баганууд: Нас зэрэг, Эр (тоо + progress), Эм (тоо + progress), Нийт
- Progress bar-ын өргөн нь тухайн мөрийн тоо / хамгийн их тоотой мөрийн тоогоор тооцоолно
- Эр: #1890ff, Эм: #eb2f96

### 4-р мөр (2 баганатай)

**Үүлдрээр** ба **Бүлгээр** — сайжруулсан хүснэгт

Одоогийн Table-д нэмэлт:
- Эр/Эм баганад тоон утгын хажууд жижиг inline Progress bar
- Нийт баганад bold + өнгөт Tag
- null id-тай мөрийг (Бүлэггүй/Үүлдэргүй) саарал italic-аар ялгах
